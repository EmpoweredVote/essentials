Follows `scripts/md_executives_headshots.py` exactly except for the OFFICIALS list (141 delegates), TMP_DIR path, STORAGE_PREFIX, and source URL pattern.

Output: 1 Python script + best-effort headshot rows in `essentials.politician_images` for as many of the 140 active delegates as the script can successfully process.
