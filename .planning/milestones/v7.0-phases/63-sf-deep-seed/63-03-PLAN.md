**IMPORTANT — resize step (the skill does NOT resize automatically; per `feedback_headshot_image_sizing.md` and `feedback_headshot_resize_no_distort.md`):**

After the skill downloads to TMPFILE and BEFORE upload, run an explicit PIL crop-then-resize. Crop to 4:5 ratio FIRST (eyes at ~1/3 from top per `feedback_headshot_cropping.md`), THEN resize to 600x750 Lanczos q90. Never stretch directly.

```bash
python3 - <<'EOF'
from PIL import Image
src = "${TMPFILE}"
img = Image.open(src)
w, h = img.size
target_ratio = 4 / 5
current_ratio = w / h
if current_ratio > target_ratio:
    new_w = int(h * target_ratio)
    left = (w - new_w) // 2
    img = img.crop((left, 0, left + new_w, h))
elif current_ratio < target_ratio:
    new_h = int(w / target_ratio)
    img = img.crop((0, 0, w, new_h))
img = img.resize((600, 750), Image.LANCZOS)
img.save(src, "JPEG", quality=90)
print(f"Resized to {img.size}")
EOF
```

Then upload. Confirms uploaded files are exactly 600×750.
