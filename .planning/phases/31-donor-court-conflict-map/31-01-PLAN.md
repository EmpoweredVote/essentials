## Legal Occupation Keywords (use as filter)
```
attorney, lawyer, counsel, partner, esquire, esq, solicitor, litigator, paralegal,
public defender, district attorney, prosecutor
```
NOTE: Do NOT include the bare word "associate" — it has too many non-legal collisions in the LA Socrata data. If a candidate's filings show high "associate" volume, log it but exclude.

## Field Names (CRITICAL — pitfall #3 in 31-RESEARCH.md)
- Employer: `COALESCE(c.raw_record->>'contributor_employer', c.raw_record->>'con_empr', '')` — NEVER plain `con_emp`
- Occupation: `COALESCE(c.raw_record->>'contributor_occupation', c.raw_record->>'con_occp', '')`
- Donor name: `COALESCE(c.raw_record->>'contributor_name', c.raw_record->>'con_name', c.donor_name_normalized)`

## Top-15% Cumulative Threshold (CRITICAL — pitfall #2 in 31-RESEARCH.md)
- "Top 15%" = donors whose contributions cumulatively make up the first 15% of grand_total raised, sorted desc by total.
- Single mega-donor at 20% IS included (one person exceeds the threshold).
- Empty list is a valid honest output.

1. **For each non-skipped candidate**, query `transparent_motivations.contributions` JOIN `transparent_motivations.politician_sources` WHERE `ps.essentials_politician_id = $1` AND `ps.research_status = 'confirmed'`. Pull rows with the donor name, employer, occupation, and amount using the COALESCE patterns in `<reference_data>`.

2. **Filter to legal professionals** using the LEGAL_OCCUPATION_KEYWORDS list in `<reference_data>`. Occupation match = case-insensitive substring match on the COALESCE'd occupation value. Apply the occupation-first rule from CONTEXT: a donor with a legal occupation is included regardless of employer. If the donor has a legal occupation but blank employer, treat their own normalized name as `raw_firm_name` (solo attorney path).

3. **Normalize and group firms** with this logic in a `normalizeFirmName(raw)` helper:
   - lowercase
   - strip suffixes via regex: `\b(llp|lp|llc|inc|corp|pc|apc|pllc|ltd|p\.c\.|a\.p\.c\.)\b\.?`
   - replace `&`, `,` with space
   - collapse whitespace, trim
   Group all donor rows by normalized name; sum amounts; collect distinct occupations into `occupations_seen[]`; count distinct donor names into `donor_count`.

4. **Fuzzy dedup pass** using `fastest-levenshtein`'s `distance(a, b)`:
   - For every pair of normalized firm names with the same first letter, compute Levenshtein distance.
   - If `distance <= 3` AND both have `total_donated > 0`, merge the smaller-total entry into the larger and set `is_fuzzy_match=true` on the survivor.
   - If `distance` is between 4 and 6, do NOT merge but tag both with `needs_review=true` and emit a stdout warning naming both raw_firm_names — Chris will review during Plan 02.

5. **Compute 15% threshold per candidate**:
   - `grand_total` = SUM(amount) across ALL confirmed contributions for the candidate (not just legal-professional donors). Query separately and log it.
   - Sort grouped firms desc by total_donated.
   - Walk the sorted list with a running cumulative sum. Include each firm whose pre-add cumulative is `< 0.15 * grand_total`. (This matches the cumulative-dollar definition: include the firm that crosses the line; stop at the next.)
   - `threshold_cutoff` = `0.15 * grand_total` (log it for transparency).

6. **Write `court-research-input.json`** matching the exact shape in `<reference_data>`. Use `JSON.stringify(arr, null, 2)`. If a candidate has zero legal-professional donors above the threshold, include them in the JSON with `firms: []` so Plan 02 has an explicit zero-state record.

7. **Log a summary block at the end**: per candidate — grand_total, threshold_cutoff, total firms after dedup, firms with needs_review. Plus an overall "wrote N candidates, M unique firms, K needs_review".

ANTI-PATTERNS to avoid (per 31-RESEARCH.md):
- Querying `con_emp` (wrong field name; will return zeros).
- Including the word "associate" in the keyword filter.
- Treating "top 15%" as "top 15% of donors by count" (it's cumulative dollar).
- Forgetting the empty-firms zero state for candidates with no qualifying donors.
