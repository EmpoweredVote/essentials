## API Response Shape
```json
{
  "politician_id": "uuid",
  "is_city_attorney_candidate": true,
  "conflicts": [
    {
      "id": "uuid",
      "firm_name": "morrison foerster",
      "raw_firm_name": "MORRISON & FOERSTER LLP",
      "total_donated": 9500.00,
      "is_top_15pct": true,
      "court_appearances_found": 3,
      "court_appearances_capped": false,
      "case_types": ["Civil"],
      "first_appearance_date": "2022-03-15",
      "last_appearance_date": "2024-11-01",
      "source_urls": ["https://www.lacourt.ca.gov/casesummary/v2web3/?caseNumber=22STCV09876"],
      "has_conflict": false,
      "recusal_found": false,
      "conflict_note": null,
      "research_date": "2026-05-09",
      "researcher_notes": "Searched 'Morrison Foerster LLP' as Civil party..."
    }
  ],
  "summary": {
    "total_firms": 1,
    "conflicts_count": 0,
    "recusals_count": 0,
    "pending_count": 0
  }
}
```

`is_city_attorney_candidate` is taken from any conflicts row (they all share the same value for a given politician). If the conflicts array is empty, the API derives the flag by querying `essentials.politicians.office_title` and matching `lower(office_title) LIKE '%city attorney%'`.

## isLegalCandidate Detection (existing in CandidateProfile.jsx)
```jsx
const isLegalCandidate = (
  polResult?.district_type === 'JUDICIAL' ||
  polResult?.district_type === 'NATIONAL_JUDICIAL' ||
  (polResult?.office_title || '').toLowerCase().includes('city attorney') ||
  (polResult?.office_title || '').toLowerCase().includes('district attorney')
);
```

## isCityAttorney Derivation (new — pass as prop)
```jsx
const isCityAttorney = (polResult?.office_title || '').toLowerCase().includes('city attorney');
```
