**Step 2: For each senator (loop over the 35 in external_id order DESC, i.e., District 35 down to District 1, OR ascending — pick one and stick with it):**

Per-senator workflow (mirrors Phase 51-03 mechanics but no per-photo approval gate):

1. **Compute district_number** from `geo_id`: `district_num = int(geo_id[2:])` — e.g., '23009' → 9.

2. **Fetch the senator profile page**:
   ```bash
   curl -A "EmpoweredVoteHeadshotImporter/1.0 (tjefferson@empowered.vote)" \
        -L -o "/tmp/me_senate_d{N}_profile.html" \
        "https://legislature.maine.gov/senate/district{N}"
   ```
   Replace `{N}` with the district number (no zero padding for the URL path — e.g., `district9` not `district009`).

3. **Extract the direct image URL** by scanning the HTML for an `<img>` tag pointing to `/uploads/visual_edit/`. Use grep/regex:
   ```bash
   grep -oE 'https?://(www\.)?legislature\.maine\.gov/uploads/visual_edit/[^"'"'"']+\.(jpg|jpeg|png|webp)' /tmp/me_senate_d{N}_profile.html | head -1
   ```
   If no image URL is found, OR the page returns 404 / 5xx, this senator goes into the GAP list (reason: "no image on profile page" or "profile page returned {status}"). Skip to the next senator.

4. **Download the image to a temp path**:
   ```bash
   curl -A "EmpoweredVoteHeadshotImporter/1.0 (tjefferson@empowered.vote)" \
        -L -o "/tmp/{politician_id}-raw.{ext}" \
        "{image_url}"
   ```
   Note: file extension matches the source (.jpg, .png, or .webp). PIL handles all three natively.

5. **Headspace + composition check** — open the raw image with PIL and confirm:
   - Resolution is at least 600x750 (or larger — anything smaller cannot produce a quality 600x750 crop). If raw resolution is smaller, GAP it (reason: "image below 600x750 usable resolution"). Skip to next senator.
   - Single subject (no group shots). If group shot, GAP it (reason: "group photo on profile page; no individual portrait available").
   - No superimposed text, campaign banners, or graphics over the face. If present, GAP it (reason: "image contained banner/text over face").
   - Eyes at roughly 1/3 from the top of the headshot's usable area (post-crop). If framing is unrecoverable (face filling whole frame, or off-center extreme), GAP it.

6. **Crop to 4:5 aspect, THEN resize to 600x750** with PIL LANCZOS quality=90:
   ```python
   from PIL import Image
   img = Image.open("/tmp/{politician_id}-raw.{ext}")
   img = img.convert("RGB")  # safe for webp/png input
   w, h = img.size
   target_ratio = 4/5  # width/height
   if w/h > target_ratio:
       # too wide — crop horizontally (center)
       new_w = int(h * target_ratio)
       left = (w - new_w) // 2
       img = img.crop((left, 0, left + new_w, h))
   else:
       # too tall — crop vertically (preserve top for face)
       new_h = int(w / target_ratio)
       img = img.crop((0, 0, w, new_h))
   img = img.resize((600, 750), Image.LANCZOS)
   img.save("/tmp/{politician_id}-headshot.jpg", "JPEG", quality=90)
   ```

7. **Upload to Supabase Storage**:
   ```bash
   export SUPABASE_SERVICE_ROLE_KEY="eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpc3MiOiJzdXBhYmFzZSIsInJlZiI6Imt4c2R6YW9qZmFpYmh1em1jbGZxIiwicm9sZSI6InNlcnZpY2Vfcm9sZSIsImlhdCI6MTc1MDM2NTEwMywiZXhwIjoyMDY1OTQxMTAzfQ.6cZBx-L-pFiNOf3r6c9xolq2RHZT3pBsVdZxsVqYnYo"

     -H "Authorization: Bearer ${SUPABASE_SERVICE_ROLE_KEY}" \
     -H "Content-Type: image/jpeg" \
     -H "x-upsert: true" \
   ```
