**Step 1: Build the Tier 2 work list:**

```sql
SELECT p.id, p.full_name, p.urls, g.name AS city
FROM essentials.politicians p
JOIN essentials.offices o ON o.id = p.office_id
JOIN essentials.chambers ch ON ch.id = o.chamber_id
JOIN essentials.governments g ON g.id = ch.government_id
LEFT JOIN essentials.politician_images pi ON pi.politician_id = p.id
WHERE g.state = 'TX'
  AND g.name IN ('Murphy', 'Celina', 'Prosper', 'Richardson')
  AND p.is_active = true
  AND p.is_vacant = false
  AND pi.id IS NULL
  AND p.photo_origin_url IS NULL
ORDER BY g.name, o.title;
```

Expected count: up to ~23 politicians (Murphy 5, Celina 4 [pending Mayor/P4/P5 stubs not in DB], Prosper 7, Richardson 7).

**Step 2: Process each politician city-by-city using the skill.** Per-city source priority:

- **Murphy:** Start at the bio URL in `politicians.urls` (e.g., `murphytx.org/Directory.aspx?EID=X`). Individual EID pages MAY have photos. Fallback: Ballotpedia.
- **Celina:** Politicians' `urls` likely point to `celina-tx.gov/Directory.aspx?did=4` (shared council directory page, not individual). Browse that page for per-member portraits. CRITICAL: use `celina-tx.gov` (hyphenated) — NOT `celinatx.gov`. Fallback: Ballotpedia (expect frequent fallback for Celina).
- **Prosper:** Start at the bio URL in `politicians.urls` (e.g., `prospertx.gov/directory.aspx?EID=XX`). EID pages may have photos. Fallback: Ballotpedia.
- **Richardson:** SKIP `cor.net` bio pages — they return 403 externally (confirmed Phase 14). Go DIRECTLY to Ballotpedia for all 7 Richardson politicians. If Ballotpedia has no entry, try local press as the second source.

**Hard rejection rules (apply during AskUserQuestion approval):**
- Reject any photo with superimposed text, "Re-Elect" banners, campaign logos, or graphics over the face.
- Reject any source image under 200px wide — try the next source.
- Reject any photo with additional people in the frame that cannot be cleanly cropped.
- LinkedIn is OUT OF SCOPE.

**Source ceiling for Tier 2:** Three sources allowed (city bio + Ballotpedia + local press). Tier 2 is best-effort — if all three fail, document the gap and move on. Do NOT lower the photo bar to fill a gap.

**Per-politician outcome — record in SUMMARY gap section:**
- IMPORTED: politician_id, source URL, brief description
- GAP: politician_id, full_name, city, sources tried with rejection reason

SELECT g.name AS city,
       COUNT(p.id) AS politicians_in_db,
       COUNT(pi.id) AS with_headshot,
       COUNT(p.id) - COUNT(pi.id) AS missing_headshot
FROM essentials.politicians p
LEFT JOIN essentials.politician_images pi ON pi.politician_id = p.id
JOIN essentials.offices o ON o.id = p.office_id
JOIN essentials.chambers ch ON ch.id = o.chamber_id
JOIN essentials.governments g ON g.id = ch.government_id
WHERE g.state = 'TX'
  AND g.name IN ('Murphy', 'Celina', 'Prosper', 'Richardson')
  AND p.is_active = true
  AND p.is_vacant = false
GROUP BY g.name
ORDER BY g.name;
