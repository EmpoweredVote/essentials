Primary source: sanjoseca.gov official portraits
- The sanjoseca.gov CivicPlus WAF blocks Python/curl/WebFetch (returns HTTP 403).
- Workaround (same as fremont.gov): Node.js fetch() with browser User-Agent + Referer header.
  headers: {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/120.0.0.0 Safari/537.36',
    'Referer': 'https://www.sanjoseca.gov/your-government/city-council',
  }
- The portrait URL pattern is: https://www.sanjoseca.gov/home/showpublishedimage/{id}/{timestamp}
  The {id} and {timestamp} values must be scraped from each council member's page HTML —
  they are NOT guessable from the name or district number.
- Source pages per official:
  Mayor:    https://www.sanjoseca.gov/your-government/departments-offices/mayor-and-city-council/mayor-s-office/-NID-282

                fallback: https://www.sjdistrict3.org/ (may have portrait if sanjoseca.gov page still shows Omar Torres placeholder)
  D4-D10: https://www.sanjoseca.gov/your-government/departments-offices/mayor-and-city-council/district-{N}/your-councilmember

  Matt Mahan: https://upload.wikimedia.org/wikipedia/commons/a/ae/Matt_Mahan_portrait_2025.jpg
              License: CC-BY-SA 4.0 (requires attribution — record as photo_license='cc-by-sa-4.0')
  Other council members: search Wikimedia Commons for portrait (coverage is sparse for less prominent members)

Image processing (apply to every image regardless of source):
  1. Download original
  2. Inspect dimensions — sanjoseca.gov portraits are likely ~400×600 (2:3) or similar; log actual size
  3. Crop to 4:5 ratio (never stretch): crop vertically from top if taller than 4:5; eyes should be at ~1/3 from top
  4. Resize to 600×750 using Lanczos resampling, JPEG quality 90
  5. REJECT any image with superimposed text, banners, or graphics over the face
