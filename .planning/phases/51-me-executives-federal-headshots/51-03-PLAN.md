**Per-photo workflow (repeat 8 times — one per official, in external_id order: -230001 → -230002 → ... → -230202):**

1. **Download** the source image to a temp path:
   ```bash
   curl -A "EmpoweredVoteHeadshotImporter/1.0 (tjefferson@empowered.vote)" \
        -L -o "/tmp/{politician_id}-raw.jpg" \
        "{direct_image_url}"
   ```
   The User-Agent header is REQUIRED — Wikimedia rejects empty UA requests. Government sites are less strict but include UA anyway for consistency.

   If the direct image is `.webp` (Frey's maine.gov URL ends in `.jpg.webp`), the raw file may need conversion. PIL handles webp natively if Pillow was built with WebP support. If `Image.open()` fails, run `pip install Pillow --upgrade` once and retry; otherwise re-download from the Wikipedia fallback.

2. **Headspace check** — open the raw image and confirm:
   - Eyes are at roughly 1/3 from the top.
   - Full head + shoulders are visible.
   - No additional people in the frame.
   - No campaign text/graphics over the face.
   - If framing is wrong, reject and find an alternate source (try the other priority — official vs Wikipedia — for this subject).

3. **Crop to 4:5 aspect, THEN resize to 600x750** with PIL LANCZOS quality=90:
   ```python
   from PIL import Image
   img = Image.open("/tmp/{politician_id}-raw.jpg")
   img = img.convert("RGB")  # safe for webp/png input
   w, h = img.size
   target_ratio = 4/5  # width/height
   if w/h > target_ratio:
       # too wide — crop horizontally (center)
       new_w = int(h * target_ratio)
       left = (w - new_w) // 2
       img = img.crop((left, 0, left + new_w, h))
   else:
       # too tall — crop vertically (preserve top for face)
       new_h = int(w / target_ratio)
       img = img.crop((0, 0, w, new_h))
   img = img.resize((600, 750), Image.LANCZOS)
   img.save("/tmp/{politician_id}-headshot.jpg", "JPEG", quality=90)
   ```
