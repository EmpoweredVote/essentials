**IMPORTANT — explicit PIL crop-then-resize:**

After the skill downloads to TMPFILE and BEFORE upload, run an explicit PIL crop-then-resize. **Crop to 4:5 ratio FIRST** (never change aspect ratio — only crop; eyes at ~1/3 from top; full head + shoulders visible). **THEN resize to 600x750 Lanczos q90.** Never stretch directly.

```bash
python3 - <<'EOF'
from PIL import Image
src = "${TMPFILE}"
img = Image.open(src)
# Force RGB (handles RGBA PNGs by flattening alpha on white)
if img.mode in ('RGBA', 'LA'):
    bg = Image.new('RGB', img.size, (255, 255, 255))
    bg.paste(img, mask=img.split()[-1])
    img = bg
elif img.mode != 'RGB':
    img = img.convert('RGB')
w, h = img.size
target_ratio = 4 / 5
current_ratio = w / h
if current_ratio > target_ratio:
    # Image is wider than 4:5 — crop horizontally (centered)
    new_w = int(h * target_ratio)
    left = (w - new_w) // 2
    img = img.crop((left, 0, left + new_w, h))
elif current_ratio < target_ratio:
    # Image is taller than 4:5 — crop from top (eyes at ~1/3 from top)
    new_h = int(w / target_ratio)
    img = img.crop((0, 0, w, new_h))
img = img.resize((600, 750), Image.LANCZOS)
img.save(src, "JPEG", quality=90)
print(f"Resized to {img.size}")
EOF
```
