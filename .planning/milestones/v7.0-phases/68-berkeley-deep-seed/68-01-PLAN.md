**CRITICAL — NO outSR=4326 for Socrata:**
Berkeley Open Data is Socrata and returns native WGS84 GeoJSON. Do NOT append `?outSR=4326` or any ArcGIS-style parameters — those are ArcGIS-only. The Socrata endpoint already returns WGS84 lat/lon coordinates.

**CRITICAL — Field name is `district` (lowercase string, values "1"-"8"):**
The Berkeley Socrata response uses a field named `district` (lowercase) with string values "1" through "8". Use `parseInt(String(rawDistrict ?? ''), 10)` to convert to integer. Do NOT look for `DISTRICT` (ArcGIS pattern) or `sup_dist_num` (SF DataSF pattern).
