## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-204-04 | Info Disclosure / Tampering | onCalibrate return URL | mitigate | Build `return` only from `window.location.href` (our own origin); `encodeURIComponent` it; never read a caller-supplied `return`/redirect param into the navigation |
| T-204-01 | Tampering | activeLensKey applied to grid | mitigate | Key already validated by loadLensSelection (Plan 01) + resolved against the live `lenses` list; an unknown key resolves to no lens -> Best Match, not a crash |
| T-204-02 | Injection | chip color via LensChipRow | mitigate | Colors pre-sanitized to hex by normalizeApiLens (Plan 01/02) before reaching the chip style |
| T-204-SC | Tampering | dependencies | accept | No new packages introduced |
</threat_model>
