## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| External government sites → local filesystem | Image bytes are downloaded over HTTPS from stmaryscountymd.gov and leonardtown.somd.com into the local tmp_md_local_headshots directory before processing. A malicious image (HTML disguised as JPEG, oversize file) could exploit the Pillow decoder. |
| Local script → Supabase Storage | The script uses the SUPABASE_SERVICE_ROLE_KEY to POST to /storage/v1/object/politician_photos with x-upsert: true. A compromised key could overwrite any object in the bucket. |
