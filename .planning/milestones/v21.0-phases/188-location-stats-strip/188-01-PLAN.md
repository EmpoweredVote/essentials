<objective>
Build the build-time data pipeline for the Location Stats Strip: a rebuildable Node generator that
pulls US Census ACS5 2023 population for all places + all states + the national total, and emits a
committed static JS bundle (`src/data/population.js`) keyed by FIPS. Also export the currently-private
`STATE_FIPS_TO_ABBREV` from `buildingImages.js` so the resolver (Plan 02) can invert it as the single
source of truth for state abbrev <-> FIPS.

Purpose: Satisfies D-01/D-02/D-03 and the sourcing half of STAT-02 — population is Census-derived and
keyed by geo identifier, not a hand-typed per-city table, while giving the runtime a zero-latency
synchronous in-memory lookup (no runtime Census, no CORS, no key at runtime).
Output: `scripts/gen-population.mjs`, `src/data/population.js` (committed), `STATE_FIPS_TO_ABBREV`
exported, and an `npm run gen:population` script.
</objective>

## Trust Boundaries

| Boundary | Description |

| build machine env → generator | CENSUS_API_KEY (a secret) is read from the environment at build time only |
| api.census.gov → generator | External API response (untrusted array-of-arrays) parsed into the committed bundle |
| committed bundle → runtime app | Static public non-sensitive data imported synchronously; no runtime network |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-188-01 | Information Disclosure | CENSUS_API_KEY handling in scripts/gen-population.mjs | mitigate | Key read only from `process.env.CENSUS_API_KEY`; never written to src/, never committed, never interpolated into the generated file. Verified by absence of the key in git-tracked files. |
| T-188-02 | Tampering | Census API returns a short/error body (e.g. missing-key error) | mitigate | Fail-fast row-count assertions (`places>20000`, `states>=50`) before writing; generator exits nonzero rather than writing a garbage/empty bundle (Pitfall 1). |

| T-188-SC | Tampering | npm/pip/cargo installs | accept | This plan installs ZERO packages (generator uses Node built-in fetch; Vite/Vitest already present). Package Legitimacy Gate is not applicable — no new dependency surface. |
