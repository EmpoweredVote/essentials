| GET /compass/lenses -> app | lens `color`/`name`/`description` are API-supplied strings later rendered in the UI |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-204-01 | Tampering | loadLensSelection | mitigate | Validate persisted key against `knownKeys` before returning; fall back to `'custom'` |
| T-204-02 | Injection | sanitizeLensColor / normalizeApiLens | mitigate | Reject any lens `color` not matching a hex pattern; return a safe default before it can reach an inline style |
| T-204-SC | Tampering | dependencies | accept | No npm/pip/cargo installs in this plan — no new packages introduced |
