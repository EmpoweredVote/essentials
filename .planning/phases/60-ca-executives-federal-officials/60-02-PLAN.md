`https://unitedstates.github.io/images/congress/450x550/{bioguide_id}.jpg`

These are 450×550 pixels — already 4:5 ratio. Per project headshot spec:
- NO CROP NEEDED (already correct aspect ratio)
- Resize directly to 600×750 using Lanczos algorithm, quality=90
- photo_license = 'public_domain' (unitedstates/images project is public domain)

**For each politician in the working list from Task 1:**

1. Download the image from `https://unitedstates.github.io/images/congress/450x550/{bioguide_id}.jpg`
2. Resize to 600×750 (Lanczos, q90) — NO crop step
3. Upload to Supabase Storage bucket `politician_photos` at path `{politician_id}.jpg` with x-upsert: true
4. Insert into essentials.politician_images:
   ```sql
   INSERT INTO essentials.politician_images
     (id, politician_id, storage_path, photo_license, is_primary)
   VALUES (gen_random_uuid(), '{politician_id}', '{politician_id}.jpg', 'public_domain', true)
   ON CONFLICT (politician_id) DO NOTHING;
   ```
5. Update politicians.photo_origin_url:
   ```sql
   UPDATE essentials.politicians
   SET photo_origin_url = 'https://unitedstates.github.io/images/congress/450x550/{bioguide_id}.jpg'
   WHERE id = '{politician_id}'
     AND photo_origin_url IS NULL;
   ```
6. Log result: "✓ {full_name} — uploaded {politician_id}.jpg"
