## BUG-01 — Systemic Cicero politician data corruption (Backend — Critical)

### Summary
The recent campaign finance data import corrupted the office→district chain for 50+ Cicero-sourced active politicians in California (and potentially other states). These politicians exist in the DB with `is_active: true` but have `district_type: null` and `geo_id: null` on their linked district, meaning they are invisible to the geofence-based address search.

### Confirmed broken politicians (CA, partial list — query was LIMIT 100)
Alex Padilla (US Senator), Andrea Alarcón, Andrew Gruel, Ardy Kassakhian, Bridgett Lewis, Butch Twining, Casey McKeon, Chad Williams, Cindy Allen, Claudia M. Frometa, Daniel Brotman, Daryl Supernaw, Debra Martin, Dionne Faulk, Don Kennedy, Dorothy Pemberton, Elen Asatryan, Eloy Morales, Eric Ohlsen, Gene Masuda, Gloria D. Gray, Gracey Van Der Mark, Hector Sosa, Jason Gibbs, Jason Lyon, Jennifer Perez, Jeremy Gerson, Jon Kaji, Joni Ricks-Oddie, Julia Ruedas, Justin Jones, Kristina Duggan, Laurene Weste, Letty Lopez-Viado, Lorraine Canales, Marvin Crist, Mary Zendejas, Megan Kerr, Nikki Perez, Ollie Cantos, Pat Burns, Patsy Ayala, Richard J. Loa, Rick Cole, Roberto Uranga, Sharon Kalani, Steve Madison, Suely Saro, Tamala Takahashi, Tony Wu, Vartan Gharpetian, Victor Preciado, Viviana Longoria

### Root cause (inferred)
The campaign finance import joined on politician names and clobbered `offices.politician_id` assignments or mutated `districts.district_type` for Cicero records. The politicians table also now contains **77,626 campaign finance committee records** (PACs, officeholder accounts, etc.) with `data_source: null` — these are not elected officials and should not be in this table.

### Evidence
- Alex Padilla: `source: cicero`, `is_active: true`, but `offices.title = "Councilman"`, `districts.district_type = null`, `districts.geo_id = null`
- Padilla's original Senator office for CA (NATIONAL_UPPER) does not exist — only Adam Schiff's remains
- 77,626 rows with `data_source: null` match campaign committee naming patterns (e.g. "PADILLA OFFICEHOLDER COMMITTEE, ALEX", "PADILLA FOR SENATE", etc.)

### Impact
- All listed politicians are invisible to `POST /essentials/candidates/search` (geofence query returns no match)
- All listed politicians are invisible to `GET /essentials/representatives/me`
- Name search (NavSearch) returns them but clicking through shows no office/district data

### Required fix (backend team)
1. Audit and restore correct `district_type` and `geo_id` on all affected district records
2. Restore Alex Padilla's Senator office: `district_type = NATIONAL_UPPER`, `representing_state = CA`, `geo_id = ''` (same pattern as Schiff)
3. Remove or quarantine the 77,626 campaign committee records from `essentials.politicians` — these belong in a separate campaign finance entity table, not the politicians table
4. Run the full audit query across all states (current query was CA only):

```sql
SELECT p.full_name, p.source, o.title, o.representing_state, d.district_type, d.geo_id
FROM essentials.politicians p
JOIN essentials.offices o ON o.politician_id = p.id
LEFT JOIN essentials.districts d ON d.id = o.district_id
WHERE p.source = 'cicero'
  AND p.is_active = true
  AND (d.district_type IS NULL OR d.geo_id IS NULL)
ORDER BY o.representing_state, p.full_name;
```

## BUG-02 — LocalUnavailableBanner hardcoded to Monroe County, Indiana (Frontend — High)

### Summary
Every politician profile that has no campaign finance data shows:

> "Campaign finance filings for this office are held by the **Monroe County Election Office** and are not yet available digitally."

with a mailto link to `election@co.monroe.in.us`.

This is shown for **all** politicians with no data — including California politicians, federal officials, etc.

### Location
`src/components/CampaignFinance/CampaignFinanceSection.jsx` — `LocalUnavailableBanner` component, lines 25–29.

### Root cause
The banner was written as a hardcoded placeholder for a specific Monroe County, Indiana test politician (appears during early local-politician development) and was never made generic or data-driven.

### Required fix (frontend)
The banner should show a generic message when no jurisdiction data is available, or ideally use the politician's actual `government_name` / `representing_state` to name the correct filing office.

Minimum fix — replace the hardcoded text with:

> "Campaign finance filings for this office may be held by a local or state election authority and are not yet available digitally."

And remove the `election@co.monroe.in.us` mailto link (or make it conditional on actually being a Monroe County politician).

### Acceptance criteria
- A Karen Bass profile page shows a generic unavailable message, not Monroe County
- A Traci Park profile page shows a generic unavailable message, not Monroe County
- The Monroe County email is not shown for any politician outside Monroe County, Indiana
