<objective>
Replace 7 greyscale SFUSD headshots currently in Supabase Storage with color versions.

Gap 4 root cause: The sfusd.edu staff directory serves only black-and-white photos (filenames contain 'bw' or 'B&W'). Migration 257 seeded 7 SFUSD board members and the headshot upload script in plan 87-02 uploaded whatever sfusd.edu served — all greyscale. The greyscale guard added in plan 87-04 would have caught these, but the photos are already in storage and must be replaced.

This plan requires manual photo research (checkpoint) because color source URLs cannot be automated — sfusd.edu is the only official source and it only serves B&W.

Purpose: SFUSD is the most prominent school board in the app (SF has high user volume). Greyscale headshots on a color app look broken and undermine trust in the data quality.

Output: 7 color JPEG headshots (600x750, q90) uploaded to Supabase Storage, replacing the greyscale versions. politician_images rows updated with new URLs.
</objective>
