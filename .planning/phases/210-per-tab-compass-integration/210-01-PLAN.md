<objective>
Add per-tab compass-lens memory to the four-tab officials view. Each people-tab
(Representatives, Educators, Judges) remembers its own lens; entering a tab applies
that tab's remembered lens (or its per-tab default), and an explicit lens pick
overrides and becomes that tab's remembered value for the session.

Purpose: Delivers CMP-02 (default lens shifts per tab — Judges→Judicial,
Educators→Education/fallback, Representatives→Best Match) and completes CMP-01
(compass + switcher work identically across Educators/Judges, which already share
the single `renderPoliticianCard`/`MiniCompass` path from Phase 208 D-10). This is
a pure frontend state-wiring change — no backend, no new components, no new deps.

Output: An exported pure `resolveTabLens` helper (+ unit tests) in `src/lib/compass.js`,
and the tabLensMemory state + tab-entry effect + explicit-pick interception in
`src/pages/Results.jsx`.
</objective>
