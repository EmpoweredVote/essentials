<objective>
Replace the single global `appointedFilter` state with the per-bucket `TAB_TYPE_DEFAULTS` constant map
(from Plan 01), remove the type dropdown and the name-search input from the live `FilterBar`, delete the
now-dead filter component files, and clean up the orphaned search/filter state machinery in Results.jsx.

Purpose: HDR-01 (Elected default, dropdown removed), HDR-02 (Judges tab still shows appointed judges),
and SRCH-07 (name-search box removed). Applying a fixed per-tab constant independently to each of the
three already-computed hierarchies is what structurally guarantees the Judges tab can never be emptied
by a selection from another tab — there is no longer any shared filter value to leak (D-05/D-06).

Output: a decluttered header with an honest Elected default, an Appointed Judges tab, and no name-search
box; three dead component files removed.
</objective>
