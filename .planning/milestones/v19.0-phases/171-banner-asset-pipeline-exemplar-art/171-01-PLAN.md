| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-171-02 | Tampering / Denial of Service | committed docs/scripts Windows paths | mitigate | All `C:\...` references kept inside code fences / forward slashes; verify gate `grep -c 'C:\\'` over non-fence prose returns 0 (Tailwind v4 prod build crash, memory feedback_tailwind_scans_planning_md). |
| T-171-03 | Spoofing | sourced image provenance | accept | Operator sources only real licensed Wikimedia/Unsplash photos and records attribution; AI fabrication explicitly excluded (D-09). Low risk: art assets, no executable content. |
| T-171-SC | Tampering | pip installs (Pillow/requests) | accept | No new package installs introduced — Pillow + requests already used by existing `scripts/*_wiki_photos.py`; no new dependency surface, no audit table required. |
