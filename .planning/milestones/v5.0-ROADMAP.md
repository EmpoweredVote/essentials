**Status:** ✅ SHIPPED 2026-05-18

## Overview

Build a cold-start, repeatable playbook for onboarding any US city — then prove it by taking Cambridge, MA (Harvard's backyard, RCV pioneer) to Indiana/LA caliber coverage. Every step is documented so future cities can be added without local insider knowledge.

## Milestone Summary

**Key Decisions:**

- Cambridge Mayor is appointed council-internal (is_appointed_position=true, district_type=LOCAL) — not LOCAL_EXEC, no separate election race row
- Unique index on offices.politician_id dropped for Council-Manager cities (Siddiqui holds both Mayor + Councillor office)
- MA TIGER G4110 for 58 incorporated cities; G4040 COUSUB (293 MA towns) deferred to v5.1+ Phase 48
- Phase 43 (2025 Cambridge elections) folded into 44 / N/A — historical data not surfaced in UI
- Cambridge headshot license = press_use for all (cambridgema.gov + cpsd.us bio photos are official press photos, not public domain)
- election_method is TEXT on essentials.chambers (not constrained by pg_constraint)
- government INSERT must use WHERE NOT EXISTS (essentials.governments has no unique constraint on geo_id)

**Issues Resolved:**

- Cambridge Mayor identity correction: McGovern → Siddiqui (council elected Siddiqui unanimously Jan 2026)
- Phase 43 Cambridge Elections scope decision: historical data has no user-facing impact, skip it
- Yi-An Huang headshot gap: resolved in Phase 46 (portrait from cambridgema.gov)
- pg_constraint query for election_method was a dead end — fixed in Phase 45 retrospective
- race_candidates duplicate rows risk: WHERE NOT EXISTS guard documented in elections-seed template

**Issues Deferred (Tech Debt):**

- districtScope filtering gap for at-large local offices: Cambridge councillors see unscoped compass (display quality issue only; stances are correct)
- G4040 COUSUB boundaries for 293 MA towns not loaded — Phase 48 / v5.1+
- Luisa de Paula Santos headshot: genuine unavailability (group photo ~85px/person)
