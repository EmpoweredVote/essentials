**2. Replace alphabetical sort with branch sort**

In the `useMemo` preprocessing, find the line:
```jsx
races: races.sort((a, b) => a.label.localeCompare(b.label)),
```

Replace with a branch-aware sort. Define the branch order constant near the top of the useMemo or outside the component:
```jsx
const BRANCH_ORDER = { Executive: 0, Legislative: 1, Judicial: 2 };

races: races.sort((a, b) => {
  const branchA = getBranch(a.districtType, a.cleanedPosition) ?? 'Legislative';
  const branchB = getBranch(b.districtType, b.cleanedPosition) ?? 'Legislative';
  const bScore = (BRANCH_ORDER[branchA] ?? 1) - (BRANCH_ORDER[branchB] ?? 1);
  if (bScore !== 0) return bScore;
  return a.label.localeCompare(b.label);
}),
```

The `?? 'Legislative'` fallback places unknown-branch races in the middle of the ordering. The alphabetical tiebreak preserves stable ordering for races with the same branch.
