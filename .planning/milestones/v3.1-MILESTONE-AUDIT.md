### Implicit API Contract

**applies_local/state/federal field names not enforced on frontend**
- Location: `CompassCard.jsx` lines 37–45 / `compassService.ts` lines 149–157
- Issue: The three boolean field names are hardcoded in CompassCard.jsx and emitted by compassService.ts with no shared type or validation. If the backend renames these fields, all topics would silently appear for every tier.
- Risk: Low in practice (same team, no upcoming rename). But a TypeScript interface or JSDoc type on the frontend topic object would make this explicit.
- Fix: Add a comment in CompassCard.jsx noting the contract, or extract the key names as constants.
