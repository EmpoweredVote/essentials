|----------|-------------|
| External image hosts -> local processing | sos.oregon.gov and unitedstates/images deliver untrusted bytes that are decoded by Pillow/ImageMagick. Malformed JPEG could trigger decoder bugs. |
| Local processing -> Supabase Storage public bucket | Uploaded files are publicly readable; URL is constructed deterministically from politician_id UUID. |
| politician_images.type column -> UI render path | Frontend filters on type='default'; type='headshot' causes silent invisibility. |
