**Step 1: Phase-wide coverage SQL.** Run this:

```sql
SELECT g.name AS city,
       CASE
         WHEN g.name IN ('Plano','McKinney','Allen','Frisco') THEN 1
         WHEN g.name IN ('Murphy','Celina','Prosper','Richardson') THEN 2
         WHEN g.name IN ('Anna','Melissa','Princeton','Lucas','Lavon','Fairview','Van Alstyne','Farmersville') THEN 3
         ELSE 4
       END AS tier,
       COUNT(p.id) AS politicians_in_db,
       COUNT(pi.id) AS with_headshot,
       COUNT(p.id) - COUNT(pi.id) AS missing_headshot,
       ROUND(100.0 * COUNT(pi.id) / NULLIF(COUNT(p.id), 0), 1) AS coverage_pct
FROM essentials.politicians p
LEFT JOIN essentials.politician_images pi ON pi.politician_id = p.id
JOIN essentials.offices o ON o.id = p.office_id
JOIN essentials.chambers ch ON ch.id = o.chamber_id
JOIN essentials.governments g ON g.id = ch.government_id
WHERE g.state = 'TX'
  AND g.name IN (
    'Plano','McKinney','Allen','Frisco',
    'Murphy','Celina','Prosper','Richardson',
    'Anna','Melissa','Princeton','Lucas','Lavon','Fairview','Van Alstyne','Farmersville',
    'Parker','Saint Paul','Nevada','Weston','Lowry Crossing','Josephine','Blue Ridge'
  )
  AND p.is_active = true
  AND p.is_vacant = false
GROUP BY g.name
ORDER BY tier, g.name;
```

Save the full result table for the SUMMARY.
