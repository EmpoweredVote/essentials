## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| azleg.gov (remote) → pipeline | Untrusted remote image bytes enter the pipeline over HTTP |
| pipeline → Storage/DB | Processed image + politician_images row cross into production |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-192-03 | Tampering | remote azleg.gov image fetch | mitigate | PIL `convert('RGB')` + JPEG re-encode `optimize=True` strips EXIF/embedded/stego payloads; only `image/jpeg` written to Storage |
| T-192-04 | Spoofing | politician_images ↔ politician binding | mitigate | Fetch ONLY current-roster surnames; CHAPLIK/MARSHALL/BURCH never in ROSTER; runtime UUID resolution by external_id prevents wrong-person binding on retry; Plan 03 human identity check |
| T-192-SEC | Information disclosure | SUPABASE_SERVICE_ROLE_KEY | mitigate | Key read only from gitignored `.env`; script itself gitignored (`backend/scripts/_*`); key never in a committed migration; parameterized `external_id=%s` query (no injection) |
| T-192-06 | Spoofing | wrong-but-present image | transfer | Row-count cannot detect a wrong-but-present photo; deferred to Plan 03 human-verify spot-check |
</threat_model>
