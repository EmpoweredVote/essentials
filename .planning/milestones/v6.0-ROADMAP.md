## Milestone Summary

**Key Decisions:**

- TIGER shapefile loader key varies by state: Maine uses `cd119` not `cd` — always browse TIGER2024/CD/ and check filename before configuring STATE_LAYER_ALLOWLIST
- `districts.state` casing: lowercase ('me') for COUNTY/STATE_UPPER/STATE_LOWER; UPPERCASE ('ME') for NATIONAL_LOWER — set by loader `abbrev` vs `abbrevUpper` variables
- Legislature-elected offices (AG/SoS/Treasurer) = `is_appointed_position=true` + zero race rows — research state constitution before assuming popular election
- NATIONAL_UPPER senators: uniqueness key is `(district_id, politician_id)` — `(district_id, chamber_id)` blocks 2nd senator INSERT
- Portland RCV: `election_method='rcv'` on City Council + Auditor chambers; PLACE boundary geo_id=2360545
- Multi-tier city seeding: skeletal offices first (migration), then `UPDATE SET politician_id WHERE (chamber_id, title)` — no seat_label column on essentials.offices
- House headshot thumbnails (152×202 px) upscaled to 600×750 with user approval — approach documented in template

**Issues Resolved:**

- D94 (Scott Harriman) listed as vacant in research — Wikipedia confirmed won special election 2026; corrected before migration
- Augusta address returns ME-02, not ME-01 — counter-intuitive; documented in STATE.md
- Maine US Senate Mills withdrawal (Apr 30) — excluded from race_candidates; SOS xlsx verified
- NATIONAL_UPPER uniqueness trap discovered during migration — uniqueness key pattern established for future states

**Issues Deferred:**

- Migration 182 (security_invoker views fix) — apply when DATABASE_URL accessible; no Maine functional impact
- ME G4040 COUSUB towns (293 ME towns not yet loaded) — deferred to v6.1+ (same pattern as MA Phase 48)
