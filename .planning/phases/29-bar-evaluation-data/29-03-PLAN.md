
**B. Build BarEvaluationSection.jsx**

Create `C:/Transparent Motivations/essentials/src/components/BarEvaluationSection.jsx`.

This component receives `judicialRecord` (the same shape as what Profile.jsx already has in state: `{ evaluations: [], metrics: [], disciplinary_records: [] }`) and renders a card section below JudicialCompassSection.

The component should:
1. Check if there are any evaluations from LACBA JEEC, CA State Bar, or CJP — if none, render nothing (return null).
2. Render a section with a gavel or balance-scale icon, title "Bar Evaluations & Discipline Record", and the burnt orange accent color `#c2410c` (matching JudicialCompassSection).
3. Group evaluations by source (LACBA JEEC, CA State Bar, CJP) and render each as a labeled card row.
4. If disciplinary_records.length > 0, render a sub-section "Disciplinary Records" with each record showing: record_type, record_date (formatted as human-readable), description, and a link to source_url.
5. If disciplinary_records is empty AND the candidate has a CJP evaluation row (from judicial_evaluations where source='CJP'), show the CJP evaluation rating (which will say either 'N/A ...' or have actual evaluation data).

**Component structure:**

```jsx
import React from 'react';

const ACCENT = '#c2410c';

function GavelIcon() { /* simple gavel SVG, stroke={ACCENT} */ }

function formatDate(dateStr) {
  if (!dateStr) return '';
  const iso = /^\d{4}-\d{2}-\d{2}$/.test(dateStr) ? `${dateStr}T12:00:00` : dateStr;
  const d = new Date(iso);
  if (isNaN(d.getTime())) return dateStr;
  return d.toLocaleDateString('en-US', { month: 'long', day: 'numeric', year: 'numeric' });
}

export default function BarEvaluationSection({ judicialRecord }) {
  if (!judicialRecord) return null;
  
  const { evaluations = [], disciplinary_records = [] } = judicialRecord;
  
  // Only show if there is bar evaluation data
  const lacbaEntries = evaluations.filter(e => e.source === 'LACBA JEEC');
  const barEntries = evaluations.filter(e => e.source === 'CA State Bar');
  const cjpEntries = evaluations.filter(e => e.source === 'CJP');
  
  if (lacbaEntries.length === 0 && barEntries.length === 0 && cjpEntries.length === 0 && disciplinary_records.length === 0) {
    return null;
  }
  
  return (
    <section className="mt-8" aria-label="Bar Evaluations and Discipline Record">
      {/* Header row — gavel icon + title, same style as JudicialCompassSection */}
      {/* LACBA card — show rating with badge */}
      {/* CA State Bar card */}
      {/* CJP card OR Disciplinary Records sub-section */}
    </section>
  );
}
```

For each source card, show:
- Source label (bold) + rating text
- A source link if source_url is present

For disciplinary records: show a "Disciplinary History" card with a yellow or amber left-border (not burnt orange — distinguish from compass). Each record:
- record_type as bold heading
- record_date formatted
- description text
- "Source document" link to source_url (opens in new tab)
