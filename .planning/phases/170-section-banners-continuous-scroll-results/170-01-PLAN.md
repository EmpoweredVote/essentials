<objective>
Create the reusable `SectionBanner` component (BANR-01) — a single first-party display component
that renders all three tiers (city / state / federal) as a compact full-bleed dark band with a teal
eyebrow, coral location pin, and bold title. It supports two visual variants from one prop set: an
image variant (image + mandatory dark gradient overlay) and a graceful gradient fallback variant
(tier-tinted, no image) for the common no-art case (BANR-03 component side). It also exposes
hidden/empty `stats` and `featureIcons` scaffolding slots (BANR-04).

Purpose: This component is the single divider used between City → State → Federal in Results
(Plan 170-03 wires it in) and is reused unchanged on the Elections page in Phase 172. Building it
in isolation first lets Plan 03 consume a known contract instead of exploring the codebase.

Output: `src/components/SectionBanner.jsx` (new) + `src/components/SectionBanner.test.js` (new).
</objective>
