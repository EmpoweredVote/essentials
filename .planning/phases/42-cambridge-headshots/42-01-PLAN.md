**IMPORTANT — resize step (the skill does NOT resize automatically):**
After the skill downloads the image to TMPFILE and BEFORE it uploads, add an explicit PIL resize. Between the `curl -sL ... -o "${TMPFILE}"` download and the `curl -X POST ... --data-binary` upload, run:

```bash
python3 - <<'EOF'
from PIL import Image

src = "${TMPFILE}"
img = Image.open(src)
w, h = img.size

# Crop to 4:5 ratio first (never stretch)
target_ratio = 4 / 5
current_ratio = w / h
if current_ratio > target_ratio:
    # Image is wider than 4:5 — crop width
    new_w = int(h * target_ratio)
    left = (w - new_w) // 2
    img = img.crop((left, 0, left + new_w, h))
elif current_ratio < target_ratio:
    # Image is taller than 4:5 — crop height from bottom (keep head at top)
    new_h = int(w / target_ratio)
    img = img.crop((0, 0, w, new_h))

# Now resize to 600x750 with Lanczos
img = img.resize((600, 750), Image.LANCZOS)
img.save(src, "JPEG", quality=90)
print(f"Resized to {img.size}")
EOF
```

Then proceed with the upload. This ensures uploaded files are exactly 600×750.
