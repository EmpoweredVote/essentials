**Schema (confirmed — no photo_origin_url column):**
```
id UUID, politician_id UUID, url TEXT, type TEXT, photo_license TEXT, focal_point TEXT
```

**CRITICAL: type must be 'default'** — the UI (Profile.jsx, Results.jsx) filters with `.find(img => img.type === 'default')`. Using type='headshot' or any other value causes silent invisibility.

**INSERT pattern for each official:**
```sql
INSERT INTO essentials.politician_images (id, politician_id, url, type, photo_license)
VALUES (
  gen_random_uuid(),
  (SELECT id FROM essentials.politicians WHERE external_id = {external_id}),
  'https://kxsdzaojfaibhuzmclfq.storage.supabase.co/storage/v1/object/public/politician_photos/{politician_id}-headshot.jpg',
  'default',
  '{license}'  -- e.g. 'public_domain', 'cc-by-sa-4.0'
)
ON CONFLICT DO NOTHING;
```
