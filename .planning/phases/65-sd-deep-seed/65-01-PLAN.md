**CRITICAL — outSR=4326 IS REQUIRED (unlike DataSF):**

- webmaps.sandiego.gov uses State Plane WKID 2230 (NAD 1983 StatePlane California VI, units in US survey feet) natively.
- Without `&outSR=4326`, ArcGIS returns geometries in feet (e.g. `(6295123.4, 1882346.2)`), PostGIS stores garbage, and ST_Covers returns 0 rows for any lat/lon query.
- The SF DataSF loader has NO outSR because Socrata returns WGS84 natively. DO NOT copy that — this is the wrong sibling.

async function querySdCouncilBoundaries(
  client: Client, lon: number, lat: number
